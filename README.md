# Progress Steps (Practice Project)

This is a small **practice project** built while learning JavaScript and frontend development.

The project creates an interactive **progress step component** where users can move forward and backward through steps using **Next** and **Prev** buttons.

## What It Does

* Displays multiple step circles
* Shows progress with a dynamic progress bar
* Allows navigation through steps
* Disables buttons when limits are reached

## Technologies

* HTML
* CSS
* JavaScript (Vanilla)
